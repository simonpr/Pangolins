# Pangolins
The year is 1982.  Expert systems are pretty much unheard of and simple gaming is the way of life.
Sinclair User publishes a code example, where suddenly my simple Zx Spectrum can guess the animal I am thinking about.

FFWD 43 years and I have used Google AI to not only convert this code to python, but also turn it into a web app that allows others to interact with it too.
Obviously if I allow the world to update this dataset I am going end up with countless purple penis eaters ubt that the way it goes when you release stuff into the wild
